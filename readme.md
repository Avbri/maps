
# Project Description


## DC Monument Night Walk

>This project's focus was the creation of a custom route mapped with markers and points of interest utlizing:
>Google Maps, Maps to GPX, GEOJson.io as well as leaflet and openMap libraries. 
>This route shows a one-way route designed to allow the user to see significant monuments in D.C. 
>Also included is a restaurant as a stop. This route is organized in a way that returns the user to the starting point.

>Major Funtions of note utlised in this map are the utilization of css styling sheets that build the original template, and format the style of the entire webpage.
>Additiona functions include the interaction functions created with binding the display of feature names on hover with the tooltop.
